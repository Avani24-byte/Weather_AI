 🌦 Weather AI

A simple weather web application that shows real-time weather information for any city using API-based data fetching.

 Features
- Search weather by city name
- Displays temperature, weather condition, humidity, and wind speed
- Clean and responsive UI
- Fast and lightweight frontend app

 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Weather API (OpenWeatherMap or similar)

 📁 Project Structure
Weather_AI/
│
├── index.html      # Main webpage
├── styles.css      # Styling file
├── script.js       # Weather logic and API handling
├── images/         # Icons and assets
└── .vscode/        # VS Code settings

 ⚙ How It Works
1. User enters a city name.
2. JavaScript sends a request to the weather API.
3. Weather data is fetched and displayed on the page.

 ▶ How to Run
1. Download or clone the repository
   git clone https://github.com/Avani24-byte/Weather_AI.git

2. Open `index.html` in your browser.

That's it 🎉

 🔑 API Setup (if applicable)
If the project uses an API key:
1. Get a free API key from OpenWeatherMap
2. Replace the API key inside `script.js`

Example:
const apiKey = "YOUR_API_KEY";
